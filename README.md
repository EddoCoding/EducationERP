EducationERP - Программа для автоматизирования процессов в высших учебных заведениях, переписывается с предыдущего проекта сделаного для диплома на полный MVVM с собственной библиотекой реализующий IoC и команды. 

Ссылка на него (https://github.com/EddoCoding/ERPeducation).

Пока сделано:
1. Модуль "Администрирование" (Взаимодействие с базой данных PostgreSQL, управление пользователями ERP-системы).
2. Вход по ролям с открытием нужных модулей.
3. Модуль "Приёмная кампания" (кроме электронного документооборота).

В РАБОТЕ:
1. Модуль "Деканат".

Запланировано, написание модулей:
1. Ректор.
2. Преподаватель.
3. Воинский учет.
4. Хозяйственная часть.
5. Учебный отдел.
6. Библиотека.
7. Общий функционал (Электронный документооборот, мессенджер, система уведомлений, временное храненеи файлов пока нет доступа к бд).


Алгоритм работы:
1. Скачиваете PostgreSQL (настраиваете строку подключения).
2. Запускаете ПО, заходите в модуль "Администрирование".
3. В настройках бд вводите в TextBox'ы значения строки подключения.
4. В настройках юзеров создаете юзера, выдаете доступы.
5. Перезаходите в программу, теперь так как есть бд, появиться окно с логином и идентификатором, вводим туда данные созданного пользователя.
6. Работаем в модулях, которые доступы пользователю.
